# CONTEXT+DATASETS - NEISS Injury Data

This repository contains a CONTEXT+DATASET of the NEISS Injury data, which includes information about consumer product-related injuries occurring in the United States.


- `README.md` - This file contains a description fo the files and structure of the repositry.
- `data/` - This directory contains the downloaded data files. 
- `CONTEXT.md` This file contains the context of the data.


## Data Download

The annual NEISS Data for the years 2005-2024 have been provided in the `data/raw` directory. These files were manually downloaded from the [NEISS Query Builder](https://www.cpsc.gov/cgibin/NEISSQuery/home.aspx).
